Patient Management System (PMS)
Project Overview
Patient Management System (PMS) is a desktop healthcare application developed in C++ using SFML. The system demonstrates strong Object-Oriented Programming (OOP) principles and modular software design, handling real-world healthcare tasks such as patient registration, appointment scheduling, doctor management, report handling, and payment processing.
This project showcases practical problem-solving, file-based data persistence, GUI integration, and input validation in a professional, maintainable codebase.
